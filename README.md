# PR Summarizer Bot

PR Summarizer Bot is a backend service built using Django that analyzes GitHub Pull Requests and generates structured summaries to assist developers during code reviews.

## Features
- Fetches Pull Request data using GitHub APIs
- Processes PR metadata and code changes
- Generates concise and structured summaries
- Exposes REST endpoints for integration

## Tech Stack
- Python
- Django
- GitHub REST API

## What I worked on
- Built the backend using Django
- Implemented API endpoints
- Integrated GitHub APIs for PR data retrieval
- Handled request processing and response generation
